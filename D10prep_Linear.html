Preparing For: Linear Functions and Equations

In our next class meeting we will investigate linear functions — one of the most fundamental and widely-used families of functions in mathematics. You will learn to recognize linear functions of the form \(f(x) = mx + b\), interpret their slope and intercept, construct equations of lines from given information, and solve linear equations.

This prep work is designed to take approximately 20 minutes. It reviews three algebraic skills that will be called upon directly during our class discussion. Working through these problems carefully before class will help you engage more confidently with the new material.

Objectives

By the end of our class meeting on Linear Functions and Equations, you will be able to:

  • Recognize functions of the form \(f(x) = mx + b\) as linear functions and identify the slope \(m\) and \(y\)-intercept \((0, b)\).
  • Use the slope formula to find the slope of a line connecting two points.
  • Use point-slope form to construct the equation of a line from given information.
  • Evaluate linear functions and interpret outputs in context.
  • Set up and solve linear equations arising from applied problems.

Just-In-Time Review

The three skills below are ones you have seen before. We review them here because each one appears directly in today’s class discussion. Take your time with the embedded problems — getting these right matters more than getting through them quickly.

Skill 1: Plotting Points in the Coordinate Plane

Every linear function produces a straight line when graphed in the coordinate plane. To understand what that line is doing — where it starts, how steeply it rises or falls — you need to be comfortable placing points accurately on the plane.

Recall that a point \((x, y)\) is located by moving \(x\) units along the horizontal axis and \(y\) units along the vertical axis. Points with negative coordinates move left or down respectively. The point \((0, 0)\) is the origin.

This skill connects directly to our class discussion because we will be reading information about linear functions from their graphs, and we will be verifying solutions to linear equations by checking whether a point lies on a given line.

Try It! 1: Plotting points in the Plane.

Skill 2: Finding the Slope of a Line Through Two Points

Slope measures the rate of change of a linear function — how much the output changes for every one-unit increase in the input. In our class discussion, slope will be the central object of study: we will interpret it in applied contexts, compute it from data, and use it to build equations of lines.

The slope of the line connecting two points \((x_1, y_1)\) and \((x_2, y_2)\) is:

\[m = \frac{y_2 - y_1}{x_2 - x_1} = \frac{\Delta y}{\Delta x}\]

A positive slope means the function is increasing; a negative slope means it is decreasing. A slope of zero means the function is constant. Keep an eye on the sign — it carries real meaning in context.

Try It! 2: Calculate the slope of the line between the two points.

Try It! 3: Calculate the slope of the line connecting the two points below.

Skill 3: Evaluating Functions

In our class discussion, you will encounter linear functions like \(f(x) = 3x - 7\) and be asked to find outputs for specific inputs, build tables of values, and determine when a function equals a given target value. All of these tasks require fluency with function evaluation.

Recall that \(f(a)\) means: replace every occurrence of \(x\) in the formula with \(a\), then simplify. The result is the output the function assigns to the input \(a\).

For example, if \(f(x) = 3x - 7\), then: \[f(4) = 3(4) - 7 = 12 - 7 = 5\]

Notice that evaluating a function is different from solving an equation. When we evaluate, we are given the input and we find the output. When we solve, we are given the output and we find the input. Both tasks will appear in class.

Try It! 4: Evaluate the linear function for a variety of inputs.

You’ll also be asked to solve linear equations. In doing this, remember that

  1. order of operations matters.
  2. we’ll “undo” operations in reverse-order.
  3. subtraction “undoes” addition (and vice-versa).
  4. division “undoes” multiplication (and vice-versa).

What To Expect In Class

Our class meeting opens with a real application: a gym offers memberships that include personal training sessions, and we are given pricing information for two different membership tiers. From just two data points, we will recover a complete linear model — finding the cost per session (slope) and the base membership fee (\(y\)-intercept) — and use it to answer pricing questions.

This application is a preview of something powerful: two points completely determine a line. Once you know any two input-output pairs for a linear function, you know everything about it. We will formalize this idea using the slope formula and point-slope form, then practice constructing linear equations from a variety of given information.

The second half of class turns to linear equations — situations where we know the output of a linear function and want to find the input. These arise constantly in applied settings: break-even analysis, distance-rate-time problems, and cost modeling all reduce to solving a linear equation.

By the end of class you will have the tools to take a real-world scenario, build a linear model from data, and use that model to answer questions — a complete problem-solving pipeline that will reappear throughout the course.
